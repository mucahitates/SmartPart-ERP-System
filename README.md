
# SmartPart ERP Sistemi

Bu proje, Java ile geliştirilen katmanlı mimariye sahip bir ERP (Enterprise Resource Planning) sistem simülasyonudur. 
Amaç, gerçek iş senaryolarını anlamak ve sürdürülebilir yazılım geliştirme prensiplerini öğrenmektir.

---

##  Projenin Amacı

- Java ile kurumsal yazılım mantığını kavramak  
- Katmanlı mimari yapısını deneyimlemek  
- ERP sistemlerinde ürün yönetimi süreçlerini simüle etmek  

---

##  Kullanılan Teknolojiler

- Java
- Swing
- MySQL
- JDBC
- Git & GitHub
